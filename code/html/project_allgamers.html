Allgamers

Application centrée sur le jeu vidéo

PROJET

Ce projet est réalisé en PHP avec une base de données (MySQL).

Il s'agit d'une application de communauté sociale conçue pour les joueurs de jeux vidéo.

TECHNOLOGIES

  • UX/UI sur Figma
  • PHP
  • Javascript
  • MySQL
  • Git/GitHub

COMPÉTENCES

  • Architecture MVC
  • Développement d'une base de données
  • Gestion d'un cahier des charges
  • Création d'un wireframe, d'un Mockup et d'un prototype avec Figma
  • Utilisation de GitHub
